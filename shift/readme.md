# シフト管理webアプリの作成

バイト先のシフト管理が大変そうだったので個人的に勉強含めて作ってみた。

[Readme english version](http://qiita.com)
- 使用技術
    - フロントエンド
        - ユーザー側
            - Nuxt.js + vuetify
        - 管理側
            - Next.js + Typescript + Tailwindcss
    - バックエンド
        - Django(python) + RestFramework
    - インフラ
        - Versel(フロントエンド)
        - GAE(バックエンド)
    - 選定理由
        - 勉強したものを広範囲に利用してみた。
